Todo

A todo list built on the Reaction framework. This is intended to be a minimal example.

Install and Run

You need to have the reaction repo in the same directory as the reaction-todos repo.

  1. Clone.

    bash $> git clone https://github.com/jimjh/reaction-todos

  2. Install.

    bash $> bundle install

  3. Migrate.

    bash $> rake db:migrate

  4. Run.

    bash $> bundle exec rails server

Open up two different browsers (I tried this in Chrome and Safari) and point them to http://localhost:3000. Changes made in one browser are immediately updated in the other.
